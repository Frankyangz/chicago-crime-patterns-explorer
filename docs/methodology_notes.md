# Methodology Notes

## Data Interpretation

The Chicago crime dataset represents reported incidents. It should not be interpreted as a complete measure of all crime. Changes in reporting, policing, population, public behavior, and data collection practices may affect patterns over time.

## Geographic Interpretation

Community area comparisons should be handled carefully. Raw counts may reflect population density, land area, commercial activity, tourism, or transit patterns. If possible, the final dashboard should include an adjusted metric such as incidents per population or incidents per square mile.

## Privacy and Ethics

The dashboard should focus on aggregated patterns rather than individual incidents. Visual language should avoid labeling neighborhoods as simply safe or unsafe.
